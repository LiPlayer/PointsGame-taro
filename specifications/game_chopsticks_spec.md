# 游戏开发规范：砂锅围筷子 ("Chopsticks in a Clay Pot")

本章节定义 **砂锅围筷子** 的具体设计与参数 (V2.1 Infinite Mode)。

## 1. 核心玩法逻辑 (Core Mechanics)

- **游戏模式**：**无尽积分挑战 (Infinite Score Attack)**。
- **游戏目标**：在中心旋转的"砂锅"边缘插入尽可能多的"筷子"，挑战最高分。
- **失败条件**：新射出的筷子触碰到已存在的筷子，引发碰撞。
- **胜利条件**：无（游戏以失败结束结算分数）。

## 2. 动态节奏控制 (The "Feel")

- **动态难度 (Dynamic Difficulty)**：
  - 随着分数增加，砂锅旋转的基础速度增加。
  - 变速频率 (Chaos) 随分数增加，出现更多随机的快慢变化甚至反转。
- **射击反馈**：
  - **极速射击**：筷子飞行时间应控制在瞬达感。
  - **冲击感 (Impact Shake)**：击中瞬间，砂锅本体产生微小的缩放/震动反馈。

## 3. 视觉与UI设计 (Visuals & UI)

### 3.1 3D 资产
- **砂锅 (Pot)**：
  - 深色陶土材质，带有两个**锅耳 (Handles)** 和突出的**锅沿 (Rim)**。
  - 内部为**红油 (Spicy Oil)** 表面。
  - 中心贴有**红底菱形贴纸 (Red Diamond Label)**。
- **筷子 (Chopsticks)**：
  - **锥形设计 (Tapered)**：头部细（插入端），尾部粗。
  - **朝向**：**细头朝内 (Tip Inside)** 插入砂锅，**粗头朝外**。
  - **装饰**：尾部带有**金色金属帽 (Gold Cap)**，还原高档筷子质感。

### 3.2 界面 (UI Overlay)
- **布局**：全屏 HTML 覆盖层，不干扰3D场景交互。
- **关闭按钮**：位于屏幕**左上角 (Top-Left)**。
- **分数显示**：
  - 位于屏幕**顶部居中 (Top-Center)**。
  - 采用 HTML 文本渲染，避免与 3D 场景中的筷子发生遮挡或碰撞。
  - 样式：大号数字 + "SCORE" 标签。
- **已移除**：旧版的 "Level" 关卡显示。

### 3.3 响应式布局
- 游戏容器居中于浏览器窗口，适配移动端竖屏和桌面端。
- 全屏可点击区域，确保任意位置点击都能发射筷子。
- 背景色：玫瑰粉 (Rose-50)。

## 4. 胜负结算流程

- **Fail (失败)**：
  - **触发**：两根筷子距离过近 (< 0.25 弧度)。
  - **视觉**：所有筷子产生快速物理散落效果 (爆炸)。
  - **流程**：短暂展示惨状后，跳转 Result 结算页，传递最终分数。
